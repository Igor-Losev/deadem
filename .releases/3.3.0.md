**New Features**

* `deadem`, `@deademx/dota2`: Added `StringTableType.ACTIVE_MODIFIERS` decoding into typed protos (`CModifierTableEntry` on Deadlock, `CDOTAModifierBuffTableEntry` on Dota 2).
* `@deademx/engine`: String table decoders now registered with their types. `StringTableType` carries a `lazy` flag — entries decode on first `value` access.
* `@deademx/dota2`: Added `StringTableType.ECON_ITEMS` decoding into `CSOEconItem`.
* `@deademx/engine`: String table event listeners now receive the affected entries.
* `@deademx/engine`, `deadem`, `@deademx/cs2`, `@deademx/dota2`: Exported `StringTableEvent`.
* `@deademx/engine`: Added `StringTableContainer.getByType(type)`.
* `@deademx/engine`: Added `EntityMutationEvent.getChanges(names)` — flat fields lookup.

**Improvements**

* `@deademx/engine`: BitBuffer hot-path optimization — ~5–7% parser throughput improvement.

**Documentation**

* Rewrote documentation — previously undocumented APIs now covered, game READMEs rebuilt.
