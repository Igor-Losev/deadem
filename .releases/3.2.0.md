**New Features**

* `@deademx/engine`: `Entity.getField()` now unpacks arrays and structs — `getField('m_vecFOWEntities')` → array, `getField('m_vecFOWEntities.0000')` → object, `getField('m_vecFOWEntities.0000.m_nPositionX')` → scalar.

**Bug Fixes**

* `@deademx/engine`: `unpackFlattened()`, `fieldEntries()`, and `fieldNames()` no longer emit array element counts and table presence markers.

**Deprecations**

* `@deademx/engine`: `parserThreads > 0` (multi-threaded parsing) is deprecated and will be removed in the next major release.

**Examples**

* Migrated all `unpackFlattened()` calls to `getField()` in example scripts and README.

**UI**

* `@deademx/ui`: Entity Diff tab now displays `FieldDefinition` for each field.
